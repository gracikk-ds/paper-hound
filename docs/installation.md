# Installation

## Stable release

To install ArXivPaperHound, run this command in your terminal:

```sh
uv add arxiv_paper_hound
```

Or if you prefer to use `pip`:

```sh
pip install arxiv_paper_hound
```

## From source

The source files for ArXivPaperHound can be downloaded from the [Github repo](https://github.com/gracikk/arxiv_ph).

You can either clone the public repository:

```sh
git clone git://github.com/gracikk/arxiv_ph
```

Or download the [tarball](https://github.com/gracikk/arxiv_ph/tarball/master):

```sh
curl -OJL https://github.com/gracikk/arxiv_ph/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd arxiv_ph
uv pip install .
```

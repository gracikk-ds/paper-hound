# ArXivPaperHound

![PyPI version](https://img.shields.io/pypi/v/arxiv_paper_hound.svg)
[![Documentation Status](https://readthedocs.org/projects/arxiv_paper_hound/badge/?version=latest)](https://arxiv_paper_hound.readthedocs.io/en/latest/?version=latest)

A semantic search engine for finding ArXiv academic papers based on query meaning, not just keywords.

* PyPI package: https://pypi.org/project/arxiv_paper_hound/
* Free software: MIT License
* Documentation: https://arxiv_paper_hound.readthedocs.io.

## Features

- Build the vector index from arXiv papers for a date range
- Extract relevant papers by user query
